2 МЕТОДИ РЕФАКТОРИНГУ КОДУ ПРОГРАМНОГО ЗАБЕЗПЕЧЕННЯ

2.1 Мета роботи: 
     Ознайомитися з обраними методами рефакторингу коду, застосовуючи їх на прикладах із власних програмних проєктів. Навчитися виявляти недоліки в коді та використовувати відповідні прийоми рефакторингу для підвищення його якості.

2.2  Хід роботи

1. На практичному занятті було обрано для розглядання методи рефакторингу такі як Replace Parameter with Explicit Methods, Preserve Whole Object та Separate Query from Modifier; 
2. Обрані методи рефакторингу були застосовані на прикладах программного коду з власної попередньої курсової роботи;
3. Було підготовлено презентацію для ознайомлення з методами рефакторингу та прикладах їх застосування;
4. Створено та опубліковано відеозвіт даної презентації та завантажено результати роботи на GitHub
1) Відеозвіт на YouTube: https://youtu.be/jZfaXHyldVc 
2) GitHub: https://github.com/NureKramskaOlena/arkpz-pzpi-22-9-kramska-olena/tree/main/Pract2  
5. Скриншоти презентації наведені у Додатку А.
2.3 Висновок
     Під час виконання практичної роботи було розглянуто та вивчено методи рефакторингу такі як Replace Parameter with Explicit Methods, Preserve Whole Object та Separate Query from Modifier. Було підготовлено презентацію з цієї теми, яка дозволяє ознайомитись з цима методами та прикладами їх застосування.
